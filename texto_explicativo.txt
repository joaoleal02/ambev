- Codigo que recebe uma cerveja de um pdv, e com base em sua localizacao
retorna outra cerveja similar (tanto em aspectos intrinsecos) como tambem
em seu preço
- Venda Cruzada / Venda Adicional: Desenvolver um sistema de recomendação que 
permita a venda e venda cruzada de produtos (novos produtos ou produtos já comprados) 
que possibilitará maior geração de receita.
- Windows, Linux e macOS
- Bibliotecas externas: decimal
			sklearn.cluster
			numpy
			pandas
			datetime
			random
			sklearn.metrics
			matplotlib.pyplot
			matplotlib.cm
			matplotlib.style
			sklearn.reprocessing
- Feito no Jupyter, arquivos rodados no formato csv, porem tem a opção de ser rodado com o mysql
- Copiar o arquivo do programa no Jupyter, salvar a tabela MLiHops, salvar o arquivo zipado (rascunho), e botar
a senha para descriptografar, depois upload ambos arquivos no jupyter para rodar o programa, se quiser pode mudar
o postal code para ver as recomendacoes para diferentes pdvs. Outra opcao é usar o rascunho.csv que nao esta criptografado
e que foi uploaded na secao de "Assets" do taikai.
e rodar o programa (podendo escolher o postal code do pdv que quer que seja recomendado)
